# NDArray Homework

## Instructions
Create a C++ program that demonstrates the usage of the NDArray class from `source/source_base/ndarray.h`. Complete the following tasks in order, from simplest to most complex.

## Tasks

### Task 1: 2D NDArray Creation
- Create a 2D NDArray of integers with dimensions (2, 3)
- Print its shape and size

### Task 2: Element Access and Assignment
- Create a 2D NDArray of integers with dimensions (2, 3)
- Assign values to each element using nested loops
- Print all elements

### Task 3: 3D NDArray Operations
- Create a 3D NDArray of doubles with dimensions (2, 2, 2)
- Assign values to each element using nested loops
- Print all elements

### Task 4: Copy Constructor
- Create a 2D NDArray and assign values to it
- Create a new NDArray using the copy constructor
- Print the elements of both arrays to verify the copy

### Task 5: Assignment Operator
- Create a 2D NDArray and assign values to it
- Create another NDArray and assign the first one to it
- Print the elements of both arrays to verify the assignment

### Task 6: Comparison Operators
- Create two identical 2D NDArrays
- Use the == operator to compare them
- Use the != operator to compare them
- Print the results

### Task 7: Element Access with at()
- Create a 2D NDArray and assign values to it
- Access an element using the at() method
- Print the value

### Task 8: front() and back() Methods
- Create a 2D NDArray and assign values to it
- Use front() and back() methods to get the first and last elements
- Print the values

### Task 9: Iterators
- Create a 2D NDArray and assign values to it
- Use begin() and end() iterators to iterate through all elements
- Print all elements

### Task 10: NDArray with std::string
- Create a 2D NDArray of std::string with dimensions (2, 2)
- Assign string values to each element
- Print all elements

## Submission
- Create a main.cpp file in the test_ndarray directory
- Implement all tasks
- Compile and run the program
- Submit the main.cpp file
