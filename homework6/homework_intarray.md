# IntArray Homework

## Instructions
Create a C++ program that demonstrates the usage of the IntArray class from `source/source_base/intarray.h`. Complete the following tasks in order, from simplest to most complex.

## Tasks

### Task 1: Default Constructor
- Create an IntArray object using the default constructor
- Print its size and dimension

### Task 2: 2D Array Constructor
- Create an IntArray object with dimensions (3, 2)
- Print its size, dimension, and bounds

### Task 3: 3D Array Constructor
- Create an IntArray object with dimensions (2, 3, 4)
- Print its size, dimension, and bounds

### Task 4: create() Method
- Create an IntArray object using the default constructor
- Use the create() method to resize it to (4, 3)
- Print its size, dimension, and bounds after resizing

### Task 5: Element Access and Assignment
- Create a 2D IntArray object with dimensions (2, 2)
- Assign integer values to each element
- Print all elements

### Task 6: Assignment Operator with Scalar
- Create a 2D IntArray object with dimensions (2, 2)
- Assign a scalar value to all elements
- Print all elements to verify they are set to the scalar value

### Task 7: zero_out() Method
- Create a 2D IntArray object with dimensions (2, 2)
- Assign non-zero values to all elements
- Call zero_out() method
- Print all elements to verify they are zero

### Task 8: Copy Constructor
- Create a 2D IntArray object with dimensions (2, 2)
- Assign values to its elements
- Create a new IntArray object using the copy constructor
- Print the elements of the new object to verify the copy

### Task 9: 3D Array Element Access
- Create a 3D IntArray object with dimensions (2, 2, 2)
- Assign values to all elements using nested loops
- Print all elements to verify the assignment

### Task 10: 4D Array Operations
- Create a 4D IntArray object with dimensions (2, 2, 2, 2)
- Assign values to all elements using nested loops
- Print all elements to verify the assignment
- Call zero_out() method and print again to verify

## Submission
- Create a main.cpp file in the test_intarray directory
- Implement all tasks
- Compile and run the program
- Submit the main.cpp file
