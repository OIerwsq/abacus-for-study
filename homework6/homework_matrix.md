# matrix Homework

## Instructions
Create a C++ program that demonstrates the usage of the matrix class from `source/source_base/matrix.h`. Complete the following tasks in order, from simplest to most complex.

## Tasks

### Task 1: Default Constructor
- Create a matrix object using the default constructor
- Print its number of rows and columns

### Task 2: Constructor with Dimensions
- Create a matrix object with dimensions (3, 2) and zero flag set to true
- Print the matrix

### Task 3: create() Method
- Create a matrix object using the default constructor
- Use the create() method to resize it to (4, 3) with zero flag set to true
- Print the matrix

### Task 4: Element Access and Assignment
- Create a matrix object with dimensions (2, 2)
- Assign values to each element
- Print the matrix

### Task 5: zero_out() Method
- Create a matrix object with dimensions (2, 2)
- Assign non-zero values to all elements
- Call zero_out() method
- Print the matrix to verify all elements are zero

### Task 6: fill_out() Method
- Create a matrix object with dimensions (2, 2)
- Use fill_out() method to set all elements to 5.0
- Print the matrix to verify

### Task 7: Arithmetic Operators
- Create two matrix objects with dimensions (2, 2)
- Assign values to both
- Perform addition, subtraction, and multiplication by a scalar
- Print the results

### Task 8: Matrix Multiplication
- Create two matrix objects: one with dimensions (2, 3) and another with dimensions (3, 2)
- Assign values to both
- Perform matrix multiplication
- Print the result

### Task 9: trace_on() Method
- Create a square matrix object with dimensions (3, 3)
- Assign values to the diagonal elements
- Compute the trace using the trace_on() method
- Print the result

### Task 10: transpose() Function
- Create a matrix object with dimensions (2, 3)
- Assign values to each element
- Compute its transpose using the transpose() function
- Print both the original and transposed matrices

## Submission
- Create a main.cpp file in the test_matrix directory
- Implement all tasks
- Compile and run the program
- Submit the main.cpp file
