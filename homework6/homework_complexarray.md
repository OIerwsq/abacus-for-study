# ComplexArray Homework

## Instructions
Create a C++ program that demonstrates the usage of the ComplexArray class from `source/source_base/complexarray.h`. Complete the following tasks in order, from simplest to most complex.

## Tasks

### Task 1: Default Constructor
- Create a ComplexArray object using the default constructor
- Print its size and bounds

### Task 2: Constructor with Bounds
- Create a ComplexArray object with bounds (3, 2)
- Print its size and bounds

### Task 3: create() Method
- Create a ComplexArray object using the default constructor
- Use the create() method to resize it to (4, 3)
- Print its size and bounds after resizing

### Task 4: Element Access and Assignment
- Create a ComplexArray object with bounds (2, 2)
- Assign complex values to each element
- Print all elements

### Task 5: zero_out() Method
- Create a ComplexArray object with bounds (2, 2)
- Assign non-zero values to all elements
- Call zero_out() method
- Print all elements to verify they are zero

### Task 6: negate() Method
- Create a ComplexArray object with bounds (2, 2)
- Assign complex values to each element
- Call negate() method
- Print all elements to verify they are negated

### Task 7: Arithmetic Operators
- Create two ComplexArray objects with bounds (2, 2)
- Assign values to both
- Perform addition, subtraction, and multiplication by a scalar
- Print the results

### Task 8: Compound Assignment Operators
- Create two ComplexArray objects with bounds (2, 2)
- Assign values to both
- Use compound assignment operators (+=, -=, *=)
- Print the results after each operation

### Task 9: dot() Function
- Create two ComplexArray objects with bounds (2, 2)
- Assign values to both
- Compute their dot product using the dot() function
- Print the result

### Task 10: abs2() Function
- Create a ComplexArray object with bounds (2, 2)
- Assign complex values to each element
- Compute the sum of absolute squares using the abs2() function
- Print the result

## Submission
- Create a main.cpp file in the test_complexarray directory
- Implement all tasks
- Compile and run the program
- Submit the main.cpp file
